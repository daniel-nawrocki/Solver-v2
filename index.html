What This App Does

Timing Solver lets you import drill-hole coordinates, define directional timing relationships, evaluate timing combinations, and prepare a print-ready diagram preview.

Quick Start
  1. Import a CSV and map the collar or toe columns.
  2. Select an origin hole.
  3. Draw H-H, R-R, and Offset relationships.
  4. Set H2H, R2R, and Offset timing ranges.
  5. Run the solver and choose a preview result.
  6. Open print preview and print the page.

Importing a CSV

Open Import, choose the CSV file, then map the columns used for X, Y, optional Toe X/Toe Y, and hole ID. The app automatically guesses common headers when possible.

Use collar coordinates for the default diagram view. If toe columns are provided, you can switch between collar and toe in the View menu later.

Navigating the Diagram

Pan and Zoom

Drag the canvas to pan. Use the mouse wheel to zoom in and out around the pointer.

Rotate

Hold Ctrl and use the mouse wheel to rotate. Scroll up and down to rotate in opposite directions.

View Tools

Use Fit View, coordinate view switching, relationship visibility, and the rotation buttons in the View panel.

Setting Origin and Relationships

Choose the active tool from the bottom toolbar, then click holes on the diagram to build the timing graph.

Origin
Sets the reference hole at time zero.
H-H
Creates a hole-to-hole directed delay relationship.
R-R
Creates a row-to-row directed delay relationship.
Offset
Creates a directed offset relationship using the global offset range.

All holes must be reachable from the origin using directed relationships before the solver can return valid timing combinations.

Timing Ranges and Solving

Open Timing to adjust Hole-to-Hole, Row-to-Row, and Offset min/max ranges. These ranges define the values the solver explores.

After clicking Find Timing Combinations, select a result from the list to preview firing times and the timing summary overlay on the diagram.

Preview Readout

H2H and R2R show the selected delay values. Peak(8ms) shows the most holes firing in any 8 ms window.

Print Preview and Printing

Use Export PDF to open the full-screen print workspace. The paper border shows the page area for a standard sheet.

Print-only Controls

Zoom and pan inside print preview, change text size, toggle greyscale or color, and show or hide relationship lines without affecting the main editor.

Fit and Back

Use Fit Page to reset the preview framing, Back to return without printing, or Print to launch the print dialog and return to the main screen.

Troubleshooting

No timing results

Check that an origin is set, every hole is reachable, and the timing graph does not contain conflicting derived times.

Import problems

Verify that the selected X and Y columns contain valid numeric coordinates and that both Toe columns are either selected together or left blank.

Print output not right

Open print preview, fit the page, and adjust text size or relationship visibility there. Those changes only apply to the print view.
