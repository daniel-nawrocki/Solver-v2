At A Glance

Home and mode structure

The home screen is the front door to Daniel Fire. From there, the centered mode toggle switches between Diagram and Timing inside the same shared planner.

Diagram Mode

Import, edit, annotate, and prepare the shot

Import

  • Map collar coordinates, toe coordinates, and hole id.
  • Optional fields include angle, bearing, and depth.
  • Missing or invalid angles default to 0.
  • Angle 0 is treated as no real angle for labels and arrows.

View

  • Toggle grid, angle, bearing, bearing arrows, and depth.
  • Switch between Collar and Toe coordinate views.
  • Use fit and rotation controls to frame the shot quickly.

Toolkit

  • Select for click and shift multi-select
  • Box for drag selection
  • Polygon for shape-based selection
  • Markup and Text for annotations that also print

Shot menu

Use the Shot menu for shot number, location, bench, default hole diameter, and face/interior pattern values.

  1. Enter all four pattern values.
  2. Click Assign Face.
  3. Draw a polygon around the face holes.
  4. Click Apply Pattern.

The face set persists until you redefine it, clear it, or import a new shot.

Properties menu

Apply per-hole or multi-hole changes for burden, spacing, diameter, angle, bearing, depth, and stem height.

Use Apply Default Diameter to push the shot default diameter into the selected holes, or the full shot if nothing is selected.

Volume

Whole-shot cubic yards and tonnage

Calculation

Volume is calculated per hole, then summed across all valid holes.

(burden × spacing × depth) / 27 = cubic yards
cubic yards × rock density = tons

Current behavior

  • Density defaults to 2.3 tons / cubic yard.
  • Only holes with valid burden, spacing, and depth are included.
  • The menu shows whole-shot included holes, cubic yards, and tons.
  • Changing burden, spacing, or depth updates the totals.
Timing Mode

Build relationships and solve timing combinations

Standard timing flow

  1. Select an origin hole.
  2. Draw H-H, R-R, and offset relationships.
  3. Enter timing ranges.
  4. Run the solver.
  5. Choose a result to preview or simulate.

Shared import advantage

You do not need to import again after preparing the shot in Diagram. Timing uses the same shared project holes and can be opened directly from the planner toggle.

Print Preview

Build a print set one page tab at a time

Core controls

  • Back
  • Print
  • Fit Page
  • Add Page
  • Add Timing Page when timing results exist

Diagram-specific controls

  • Text size
  • Color mode
  • Angle visibility
  • Bearing visibility
  • Depth visibility
  • Bearing arrow weight and length

Label editing

  • Use Edit Labels to drag label boxes.
  • Use Reset Labels to restore defaults.
  • Label moves are stored only for the current print session.

Important print rule

Each print tab is its own page. The current print CSS and page-output flow are intended to produce one printed sheet per tab without extra blank pages between them.
